# HTML shell + CSS (two panels)

## DESCRIPTION

Build the two-panel UI shell per `docs/specs.md`.

`templates/index.html`:
- **Panel 1 — App Canvas** (`flex-1`): scrollable chat message list, a
  bottom-pinned input bar with a Send button, and a "Simulate API Failure" button
  in the top-right corner.
- **Panel 2 — Control + Trace** (`360px`): title `phones.llmfn.com`, a current-layer
  dropdown (1–8), "Reset to defaults" button, "Copy trace as JSON" button, and a
  container for the vertical trace timeline.
- Load JS with `<script type="module" src="/static/js/app.js">`.

`static/css/styles.css`:
- Flexbox two-column layout (canvas `flex-1`, panel fixed `360px`).
- Chat bubbles: user right-aligned, assistant left-aligned; an error-bubble style.
- Product-card style (used later by Layer 1).
- Trace rows with status colors: green=success, yellow=fallback, red=error,
  grey=skipped; expandable input→output detail.

## ACCEPTANCE CRITERIA

- [x] `templates/index.html` renders both panels with all listed controls
- [x] `static/css/styles.css` lays out the two columns and styles bubbles + trace
      rows with the four status colors
- [x] Page loads `static/js/app.js` as an ES module
